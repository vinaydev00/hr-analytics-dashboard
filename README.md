# 📊 HR Analytics Dashboard

> ML-powered HR analytics dashboard with real-time attrition prediction and workforce insights.

## Features
- Employee attrition risk prediction using Random Forest
- Interactive Streamlit dashboard
- Feature importance visualization
- Batch prediction for entire workforce

## Quickstart
pip install -r requirements.txt
streamlit run src/dashboard.py

## ML Model
Uses Random Forest Classifier trained on synthetic HR data.
Predicts attrition risk based on satisfaction, salary, promotion history and more.
